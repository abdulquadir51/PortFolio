I'm **Abdul Quadir**, an **Electrical & Electronics Engineering** student at **Lovely Professional University**, specializing in VLSI design, embedded systems, and industrial automation.

---

## ✨ Features

- 🌙 **Dark / Light Mode** toggle with smooth transitions
- 🎨 **Glassmorphism** design with gradient accents
- ⌨️ **Typed.js** animated hero text
- 🎭 **AOS.js** scroll animations
- 📱 **Fully Responsive** — mobile, tablet, and desktop
- 📬 **Contact Form** with Formspree integration
- 🔝 **Scroll-to-top** button with progress indicator
- 🃏 **Interactive Cards** with tilt hover effects
